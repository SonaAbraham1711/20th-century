# 20th Century Scrape

This repo contains a Jupyter notebook that scrapes the Wikipedia page **Key events of the 20th century** 
using Selenium and saves the scraped content to a `.txt` file.

## Files
- `20th_century_scrape.ipynb` — scraping notebook
- `key_events_20th_century.txt` — scraped Wikipedia content (saved in working directory)

## How to run
1. Open the notebook in JupyterLab
2. Run cells in order (Cell 1 → Cell 2 → Cell 3)
3. The output text file is created in the working directory
